# codex-demo-app

A React + Vite landing page starter with:
- Sticky navbar
- Hero section with CTA buttons
- Contact form

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages deployment

1. Update `homepage` in `package.json` with your GitHub username.
2. If your repository name differs, update `base` in `vite.config.js`.
3. Deploy:

```bash
npm run deploy
```

This publishes the built app in `dist/` to the `gh-pages` branch.
